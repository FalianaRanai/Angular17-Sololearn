2. Liaison de propriété avec les crochets [] :

Les crochets ([]) permettent de lier des propriétés d'éléments HTML à des propriétés ou expressions du composant TypeScript . Par exemple, si vous avez une propriété urlImage dans votre composant TypeScript et que vous souhaitez lier cette propriété à la source d'une balise dans le template HTML, vous pouvez le faire comme ceci :

TYPESCRIPT:

      
        {{codeBlockTypescript}}

HTML:

      
        {{codeBlockHTML}}
