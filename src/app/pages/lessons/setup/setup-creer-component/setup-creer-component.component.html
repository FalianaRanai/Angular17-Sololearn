7. Créer un component

Pour créer un component angular il faut faire:

      
        {{"ng generate component nom_du_component"}}

NB: Il faut enlever standalone et imports dans le .ts et importer le component dans le module (de base: app.module.ts)

Une autre solution serait de faire directement:

      
        {{"ng generate component nom_du_component --standalone false"}}
