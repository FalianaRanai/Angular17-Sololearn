8. Créer un module

Pour bien structurer un projet Angular, il faut répartir les composants dans différents modules.

Les composants liés à la fonctionnalité Setup se trouveront dans le SetupModule

Pour créer un module angular il faut faire:

      
        {{"ng generate module nom_du_module"}}

NB: On déclare les composants dans declarations mais on doit aussi les exporter dans exports

Ensuite, dans app.module.ts, on importe le module nouvellement créé dans imports 

      
        {{ exempleModule }}
