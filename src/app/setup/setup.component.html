Setup
1) Installer angular

{{ "Il faut installer node. Ensuite faire " }} {{ "npm install -g @angular/cli" }}

2) Créer un projet

{{ "Pour créer un projet angular il faut faire: " }} {{ "ng new nom_du_projet" }}

3) Exécuter l'application

{{ "Pour exécuter un projet angular il faut faire: " }} {{ "ng serve --open" }}

Si il y a une erreur c'est probablement qu'il faut installer les package de package.json en faisant: {{ "npm install" }}

4) Mise en place de routing

{{ "Pour exécuter un projet angular il faut faire: " }} {{ "ng serve --open" }}

Dans app.routing.module.ts, on définit les routes comme ceci:

{{ "const routes: Routes = [ " }}

{{ "{ path: '', title: 'Titre de la page', component: HomeComponent }, " }}

{{ "{ path: 'about', component: AboutComponent, children: [ { path: '', title: 'Titre de la page', component: HomeComponent } ] }," }}

{{ "];" }}

5) Component parent et enfant

Pour afficher un component enfant défini grêce à children dans path de app.routing.module.ts dans un component parent, il faut ajouter

<{{ "router-outlet" }}><{{ "/router-outlet" }}> dans le html

6) Créer un service

{{ "Pour exécuter un service angular il faut faire: " }} {{ "ng generate service nom_du_service" }}

Un service sert à prendre des données d'API par exemple

7) Créer un service

{{ "Pour exécuter un component angular il faut faire: " }} {{ "ng generate component nom_du_component" }}

Il faut enlever standalone et imports dans le .ts et importer le component dans le module (de base: app.module.ts)
